## TL;DR

This is my personal developer portfolio built with React.js! It showcases some of my favorite GitHub projects, my technical skills, and allows people to contact me directly. 

This template was originally adapted from [Soumyajit4419's Portfolio](https://github.com/soumyajit4419/Portfolio).

## Built With

This project is built using these awesome technologies:

- **React.js** (Frontend)
- **Node.js / Python Flask** (Backend Email Handling)
- **CSS3** (Styling)
- **Vercel** (Deployment)

## Features

**📖 Multi-Page Layout**
**🎨 Interactive React-Bootstrap UI with smooth animations**
**📱 Fully Responsive for Mobile & Desktop**
**📧 Secure Contact Form powered by EmailJS and a Python backend**

## Getting Started

Clone down this repository. You will need `node.js` and `python` installed globally on your machine.

### Frontend Installation

1. Install Node dependencies: `npm install`
2. Start the React development server: `npm start`
3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Backend Installation (For Contact Form)

1. Navigate to the server folder: `cd server`
2. Set up a virtual environment: `python3 -m venv venv`
3. Activate it: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
4. Install Python dependencies: `pip install -r requirements.txt`
5. Start the Flask server: `python app.py`

*Note: You must have an `.env` file in the `server` directory with your EmailJS credentials for the contact form to work.*
